# Blog Admin - TipTap Rich Text Editor Guide

## What's New

The blog editor now uses **TipTap**, a professional WYSIWYG (What You See Is What You Get) editor built on ProseMirror, providing the best text editing experience similar to Google Docs and Medium.

## Editor Toolbar

### Text Formatting
- **B** - Make text bold (Ctrl+B)
- **I** - Make text italic (Ctrl+I)
- **<>** - Inline code formatting
- **Clear** - Remove all formatting

### Structure
- **H2** - Create a heading level 2 (appears in table of contents)
- **H3** - Create a heading level 3 (appears in table of contents)

### Lists
- **• List** - Create a bullet point list (toggle on/off)
- **1. List** - Create a numbered list (toggle on/off)

## How to Use

### Creating Content

1. **Write Naturally**
   - Click in the editor area and start typing
   - Content appears exactly as typed
   - Press Enter to create new paragraphs
   - Smooth, responsive typing experience

2. **Format Text**
   - Highlight the text you want to format
   - Click the formatting button (B for bold, I for italic, <> for code)
   - Keyboard shortcuts work: Ctrl+B, Ctrl+I
   - Button highlights to show active formatting

3. **Add Headings**
   - Click H2 or H3 to convert current line to heading
   - Or select text and click H2/H3 to apply heading
   - Headings automatically appear in the Table of Contents sidebar
   - Press Backspace to remove heading formatting

4. **Create Lists**
   - Click "• List" to start a bullet list or "1. List" for numbered
   - Press Enter to add new items
   - Press Tab to indent, Shift+Tab to outdent
   - Click the button again to exit list mode

### Real-Time Preview

The **Table of Contents** in the right sidebar automatically updates as you add H2 and H3 headings. This helps organize your content structure.

### Editing Existing Posts

1. Find the post in the "Posts" sidebar
2. Click the **Edit** button
3. Make your changes in the editor (content loads automatically)
4. Click **Update Blog** to save

## Features

- **Professional Editing**: Built on ProseMirror, the engine behind Notion and other professional editors
- **Undo/Redo**: Built-in support for undoing changes (Ctrl+Z / Ctrl+Shift+Z)
- **Copy/Paste**: Smart paste handling - formatting is preserved intelligently
- **Keyboard Shortcuts**: Full keyboard support for power users
- **Mobile-Friendly**: Touch-friendly buttons and responsive design
- **No Markdown Required**: Simple, intuitive interface

## Example Workflow

1. Type: "How to Save Money on Life Insurance"
2. Press Enter twice
3. Click H2, type "1. Choose the Right Coverage"
4. Press Enter, type content about coverage
5. Click H2 again, type "2. Compare Multiple Quotes"
6. Press Enter, type content about comparing
7. Click H2, type "3. Review Annually"
8. Press Enter, type final content
9. Watch the Table of Contents auto-populate on the right
10. Upload an image
11. Click "Publish Blog Post"

Done! Your blog is published with perfect formatting.
